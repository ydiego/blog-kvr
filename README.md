## blog
- server: koa2 + mysql + Sequelize
- admin: vue-cli3
- web: create-react-app + typescript

### run
```
// serer: 
yarn start
```
```
//admin:
cd admin & yarn serve | yarn build
```
```
//web:
cd app & yarn start | yarn build
```
